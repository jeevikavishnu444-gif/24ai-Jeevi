- Go to GitHub → New Repository
    Create your repo.
- Open the repo → click Add file → Upload files
- Drag & drop your file → click Commit changes
- (Optional) Add a README
    Click Add file → Create new file
    Name it README.md
    Type your content
    Click Commit
