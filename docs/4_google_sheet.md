- Add Trigger: Drag “On Form Submission” node and connect it as the workflow start.
- Create AI Agent: Add the AI Agent node and link it to the form trigger.
- Attach Chat Model: Under AI Agent → Chat Model, connect the Google Gemini Chat Model node.
- Configure Model: Open Gemini node → paste your Google API Key → choose your Gemini model.
- Finish Workflow: Ensure AI Agent inputs (Chat Model, Memory, Tools) are correctly linked → activate workflow.
