- Form Submission Trigger
    Add “On Form Submission” node to capture user input when the form is submitted.
- Append Data to Sheet
    Connect it to “Append row in sheet” to save the form responses into a Google Sheet.
- Google Sheets Trigger
    Add “Google Sheets Trigger (rowAdded)” to detect whenever a new row is added to the sheet.
- Send Email Automatically
    Connect the trigger to “Send a message (Gmail)” to auto-send an email whenever a new row appears.
- Activate Workflow
     Set credentials for Sheets & Gmail → test the workflow → click Activate to make the automation live.
